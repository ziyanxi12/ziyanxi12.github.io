创建我的博客过程

##步骤一
####下载所有所需的工具
下载nodejs 和 npm
使用npm下载cnpm

利用cnpm下载hexo-cil

使用-v查看是否下载成功

##步骤二
####开始进行搭建
创建一个blog文件夹，后续博客的所有操作都是存在blog文件下的

sudo hexo init 初始化hexo
hexo s 开始运行hexo，此时会获得一个端口号，本地可直接访问端口号进行上网

hexo n “XXX” new一篇名为XXX文章，并且此时会创建名为XXX的md文件
此时可以通过更改md文件，来更改网站的展示界面
vim XXX.md 利用vim更改md文件
回到blog目录下
hexo clean *更新后需要重新清理
hexo g 重新生成
hexo s 重新开始–本地可观察变化
部署到github上
在github下创建用户名.github.io仓库
cnpm install –save hexo-deployer-git 下载git
下载后可能需要设置用户名和邮箱
git config –global user.name “XXXXX”
git config –global user.email “XX@xx.com“ 输入自己的github的用户名和邮箱

需要配置blog下的_config.yml文件

底部的
deploy:
type:
更改为
deploy:
type: git
repo:https://github.com/ziyanxi12/ziyanxi12.github.io.git
branch:master

hexo d 部署到远端，推到ziyanxi12.github.io上

###下载新主题
找到github上的主题
用git clone https://xxxxxxxxx.git theme/a *克隆到theme的a目录下
更改_config.yml的theme:a

每次更新了界面都需要输入以下步骤
hexo clean 清理
hexo g 生成
hexo s 开始
hexo d 推到远端
